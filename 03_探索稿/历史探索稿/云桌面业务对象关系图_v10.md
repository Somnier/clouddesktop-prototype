# 云桌面业务对象关系图

> **设计思想复盘**：
> 1. **严守“高效沟通原则”与“分模块定接口”**：彻底删除了域内部用于表达包含、组合的“蜘蛛网”连线。实体的归属由 `namespace` 天然划分，连线仅用于表达**跨域的业务数据流转**与核心生命周期接口，极大提升了图表的沟通效率。
> 2. **去除歧义词与过度抽象**：去除了“云档案”这种容易与公有云混淆的词汇，改回“服务器端终端登记”；去除了快照实体中属于底层实现细节的“差异数据包”。
> 3. **理顺继承与业务能力**：“教师机桌面”本质上也是一种桌面，它天然拥有普通桌面的所有业务能力（如数据同步），在此基础上额外叠加了群控业务。因此采用继承（派生）关系来表达，准确切中业务逻辑。

```mermaid
classDiagram
    %% ================= 服务器域 =================
    namespace 服务器域 {
        class 服务器 {
            网络地址：IP或域名
            系统运行负载
            授权许可点数
            管理员账号：多校区
        }
        class 逻辑教室 {
            教室名称
            座位布局
        }
        class 服务器端终端登记 {
            绑定的MAC地址
            绑定的IP与计算机名
            归属的教室与座位
            实时在线状态
        }
        class 服务器端告警记录 {
            告警类型
            告警源设备：终端或服务器自身
            溯源教室与座位
            处理状态
        }
        class 服务器端基础镜像 {
            OS类型
        }
        class 服务器端快照 {
            快照名称
        }
    }

    %% ================= 辅助工具域 =================
    namespace 辅助工具域 {
        class 安装U盘 {
            终端底层系统安装包
            基础镜像包
        }
        class 教室终端清单 {
            单教室全体终端信息与排布关系
        }
    }

    %% ================= 终端基座域 =================
    namespace 终端基座域 {
        class 终端 {
            物理MAC地址：多网卡
            硬件能力规格
            外设连接状态
        }
        class 终端业务配置 {
            分配的局域网IP
            分配的计算机名
            所属教室与座位
        }
        class 终端运行状态 {
            网络在线状态
            CPU与内存负载
        }
        class 本地告警记录 {
            告警类型
            处理状态
        }
        class 本地基础镜像 {
            OS类型
        }
        class 本地快照 {
            快照名称
        }
    }

    %% ================= 终端BIOS域 =================
    namespace 终端BIOS域 {
        class BIOS引导策略 {
            可见启动项列表
            默认启动项
            数据还原模式：每次还原或保留数据
        }
    }

    %% ================= 终端桌面域 =================
    namespace 终端桌面域 {
        class 桌面 {
            桌面数据同步状态：上传中或已就绪
        }
        class 教师机桌面 {
            群控指令下发能力：唤醒或关机
        }
    }

    %% 关系
    服务器 *-- 逻辑教室
    逻辑教室 *-- 服务器端终端登记
    服务器 *-- 服务器端基础镜像
    服务器 *-- 服务器端告警记录
    服务器端基础镜像 <|-- 服务器端快照 : 派生
    终端 *-- 本地基础镜像
    终端 *-- 终端运行状态
    终端 *-- 本地告警记录
    终端 *-- 终端业务配置
    本地基础镜像 <|-- 本地快照 : 派生
    桌面 <|-- 教师机桌面 : 派生

    %% ================= 跨域核心业务接口 (数据流转) =================

    %% 交付建档与映射
    教室终端清单 ..> 服务器端终端登记 : 导入平台确立档案

    %% 资产同步与状态对齐
    终端运行状态 ..> 服务器端终端登记 : 上传
    终端业务配置 <..> 服务器端终端登记 : 双向同步
    本地基础镜像 ..> 服务器端基础镜像 : 上传
    本地快照 <..> 服务器端快照 : 双向同步
    本地告警记录 <..> 服务器端告警记录 : 双向同步

    %% 引导、环境注入与管控
    BIOS引导策略 --> 本地快照 : 决定开机加载目标与还原规则
    本地快照 --> 桌面 : 实例化为用户运行环境
    终端业务配置 ..> 桌面 : 桌面启动时读取配置获得局域网身份
```
