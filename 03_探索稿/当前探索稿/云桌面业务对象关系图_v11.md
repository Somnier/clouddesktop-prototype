# 云桌面业务对象关系图 v11

> **文档日期**：2026-03-23
> **说明**：本文件为探索推演稿，用于记录概念边界与关系取舍。
> **本版推演要点**：
> 1. **角色与账号分离**：线下服务人员不是平台权限对象，平台中的权限载体是“管理员账号”。管理员账号分为系统管理员账号与校区子管理员账号，能力集基本一致，差别主要在可见范围。
> 2. **镜像/快照同步改为“按需双向”**：终端或服务器在初始建档阶段都可能还没有任何桌面资产，因此基础镜像与快照的同步不能表达成必经的单向上传路径，而应表达为“随桌面按需同步”。
> 3. **教师机以“终端用途标记 + 特殊桌面”表达**：教师机在物理层面仍是普通终端，但在服务器端终端登记和终端业务配置中必须具备明确的教师机标记，并在桌面层叠加教室开关机能力。
> 4. **考试桌面支持物理分区式加载**：桌面并不总是由快照实例化，少数考试桌面可直接由物理分区方式加载，因此“快照 -> 桌面”关系改为“快照式桌面的数据来源”。
> 5. **交付测试是产品内能力**：硬件检测、网络测试与异常展示都不应只被理解为“终端先做、服务器只看结果”，而应支持当前教室入口和服务器平台共同发起、查看与同步。

```mermaid
    %% ================= 服务器域 =================
    namespace 服务器域 {
        class 服务器 {
            网络地址：IP或域名
            系统运行负载
            授权许可点数
        }
        class 管理员账号 {
            账号类型：系统管理员或校区子管理员
            可见校区范围
        }
        class 逻辑教室 {
            教室名称
            座位布局
        }
        class 服务器端终端登记 {
            MAC地址列表：多网卡
            当前工作MAC
            绑定的IP与计算机名
            归属的教室与座位
            终端用途标记：教师机或学生机
            实时在线状态
        }
        class 服务器端告警记录 {
            告警类型
            告警源设备：终端或服务器自身
            溯源教室与座位
            处理状态
        }
        class 服务器端基础镜像 {
            OS类型
        }
        class 服务器端快照 {
            快照名称
        }
        class 服务器端测试视图 {
            测试类型与目标范围
            平台查看与导出结果
        }
    }

    %% ================= 辅助工具域 =================
    namespace 辅助工具域 {
        class 安装U盘 {
            终端底层系统安装包
            基础镜像包
        }
        class 教室终端清单 {
            单教室全体终端信息与排布关系
            MAC地址列表与当前工作MAC
        }
    }

    %% ================= 终端基座域 =================
    namespace 终端基座域 {
        class 终端 {
            MAC地址列表：多网卡
            当前工作MAC
            硬件能力规格
            外设连接状态
        }
        class 终端业务配置 {
            分配的局域网IP
            分配的计算机名
            所属教室与座位
            终端用途标记：教师机或学生机
        }
        class 终端运行状态 {
            网络在线状态
            CPU与内存负载
        }
        class 本地告警记录 {
            告警类型
            处理状态
        }
        class 本地测试结果 {
            测试类型
            结果摘要
            生成时间
        }
        class 本地基础镜像 {
            OS类型
        }
        class 本地快照 {
            快照名称
        }
    }

    %% ================= 终端BIOS域 =================
    namespace 终端BIOS域 {
        class BIOS引导策略 {
            可见启动项列表
            默认加载目标
            数据还原模式：系统盘还原或数据盘保留
        }
    }

    %% ================= 终端桌面域 =================
    namespace 终端桌面域 {
        class 桌面 {
            部署方式：快照式或物理分区式
            桌面数据同步状态：上传中或已就绪
        }
        class 教师机桌面 {
            教室开关机能力：唤醒或关机
        }
    }

    %% ================= 域内关系 =================
    服务器 "1" *-- "1..n" 管理员账号
    管理员账号 ..> 逻辑教室 : 按校区范围可见与管理
    服务器 "1" *-- "0..n" 逻辑教室
    逻辑教室 "1" *-- "0..n" 服务器端终端登记
    服务器 "1" o-- "0..n" 服务器端基础镜像
    服务器 "1" o-- "0..n" 服务器端告警记录
    服务器 "1" o-- "0..n" 服务器端测试视图
    服务器端基础镜像 <|-- 服务器端快照

    终端 "1" o-- "0..n" 本地基础镜像
    终端 "1" *-- "1" 终端运行状态
    终端 "1" o-- "0..n" 本地告警记录
    终端 "1" o-- "0..n" 本地测试结果
    终端 "1" *-- "1" 终端业务配置
    本地基础镜像 <|-- 本地快照
    桌面 <|-- 教师机桌面 : 派生

    %% ================= 跨域核心业务接口 (数据流转) =================

    %% 交付建档与导出导入
    终端 ..> 教室终端清单 : 汇总导出教室终端完整信息
    教室终端清单 ..> 服务器端终端登记 : 导入平台确立档案

    %% 初始安装与桌面资产同步
    安装U盘 ..> 终端 : 安装云桌面管理系统
    安装U盘 ..> 本地基础镜像 : 提供初始镜像来源
    本地基础镜像 <..> 服务器端基础镜像 : 随桌面按需同步
    本地快照 <..> 服务器端快照 : 随桌面按需同步

    %% 资产同步与状态对齐
    终端运行状态 ..> 服务器端终端登记 : 上传
    终端业务配置 <..> 服务器端终端登记 : 双向同步
    终端 ..> 本地告警记录 : 自动采集/比对生成
    本地测试结果 ..> 本地告警记录 : 异常时生成告警
    本地测试结果 <..> 服务器端测试视图 : 当前教室或平台发起/查看
    本地告警记录 <..> 服务器端告警记录 : 告警展示同步/处置结果回写

    %% 引导、环境注入与管控
    BIOS引导策略 --> 桌面
    本地快照 ..> 桌面
```
