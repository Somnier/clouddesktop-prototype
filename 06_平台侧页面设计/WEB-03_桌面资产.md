# WEB-03 桌面资产

> 页面角色：校区级桌面资产管理入口。
> 执行主体：系统管理员或校区子管理员。
> 数据来源：教室终端清单导入携带的桌面包、终端自行上报的桌面资产。
> 事实依据：PRD §3.1 桌面包定义、FR-04 教室终端清单导出、FR-06 教室清单导入与教室创建。

## 1. 页面职责

集中展示和管理当前校区的桌面资产。平台不直接制作或编辑桌面内容，但支持桌面包的导入、导出和删除无引用桌面。

与 WEB-02 教室详情"桌面与存储"Tab 的区别：

| 维度 | WEB-03 平台级 | WEB-02 教室级 |
| :--- | :--- | :--- |
| 组织方式 | 以桌面为第一层级 | 以教室为上下文 |
| 使用范围 | 标注桌面被哪些教室、多少终端使用 | 展示该教室内每个桌面被哪些终端使用 |
| 操作能力 | 导入桌面包、导出桌面包、删除无引用桌面 | 只读 |
| 跳转 | 点击教室名 → WEB-02 教室详情桌面 Tab | 点击终端 → WEB-02A 终端详情 |

## 3. 校区级指标

| 指标 | 说明 |
| :--- | :--- |
| 桌面总数 | 服务器端桌面条目总数 |
| 镜像 | 基础操作系统镜像数量 |
| 快照 | 快照文件数量 |
| 估算存储 | 镜像和快照的存储估算，对照服务器存储使用率 |
| 数据盘 | 创建了独立数据盘的桌面数量 |

## 4. 桌面列表

以桌面为第一层级，每张桌面卡片展示：

### 4.1 桌面基本信息
- 桌面名称、操作系统、数据盘配置、备注。
- 创建时间、更新时间。

### 4.2 使用范围
- 标注该桌面被哪些教室使用，每个教室有多少台终端在用。
- 教室名称可点击跳转 WEB-02 教室详情的"桌面与存储"Tab，在那里可进一步查看具体哪些终端在使用该桌面。
- 如果没有任何终端使用，标注"无引用"。

### 4.3 镜像与快照链
- 折叠展示该桌面的镜像→快照派生关系。每个节点显示名称和估算大小。
- 镜像和快照属于内部资产视图，展示供运维人员了解快照层级和存储分布，但不可操作（镜像和快照本身不可编辑、不可单独下发）。

## 5. 桌面包操作

### 5.1 导入桌面包

```text
┌──────────────────────────────────────────────────────────────┐
│ 导入桌面包                                                    │
│                                                              │
│ 选择桌面包文件                                                │
│ ┌── 文件选择区 ──────────────────────────────────────────┐   │
│ │  选择桌面包文件（由终端侧导出）                          │   │
│ └────────────────────────────────────────────────────────┘   │
│                                                              │
│ [开始导入]  [取消]                                            │
│                                                              │
│ 导入进度：████████████████░░░░░░░░  68%                      │
│ 正在导入镜像与快照文件…                                       │
└──────────────────────────────────────────────────────────────┘
```

- 桌面包包含桌面基本信息和实际的镜像、快照文件，因此导入是耗时操作。
- 导入过程展示进度条和当前阶段。
- 导入完成后桌面自动出现在桌面列表中。

### 5.2 导出桌面包

- 选择一个或多个桌面后点击"导出桌面包"。
- 系统打包桌面描述、镜像和快照文件生成桌面包。
- 导出同样是耗时操作，展示进度。
- 导出的桌面包可用于 U 盘携带到其他教室终端继续部署。

### 5.3 删除无引用桌面

- 点击"删除无引用桌面"后，系统列出所有无终端使用的桌面。
- 确认后删除选中桌面及其关联的镜像和快照文件，释放服务器存储。
- 有终端在用的桌面不可删除。

## 6. 与 WEB-02 教室级桌面视图的关系

WEB-02 教室详情的"桌面与存储"Tab 从教室视角展示该教室内的桌面目录，并细化到哪些终端在使用每个桌面。WEB-03 从桌面视角出发，展示每个桌面跨教室的使用范围，并提供桌面包的导入导出和清理操作。

## 7. 空态

- 校区无桌面资产时：显示"当前校区无桌面资产"，并提供导入桌面包入口。
- 某桌面无终端使用时：卡片上标注"无引用"。
