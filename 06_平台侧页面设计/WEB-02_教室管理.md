# WEB-02 教室管理

> 页面角色：平台侧教室和终端的主管理入口。
> 执行主体：系统管理员或校区子管理员。
> 数据来源：服务器端登记信息、终端上报运行状态和桌面资产、告警记录。
> 事实依据：[云桌面平台侧结论_页面范围与任务边界说明.md](../05_云桌面事实/云桌面平台侧结论_页面范围与任务边界说明.md)

## 1. 页面层级

WEB-02 包含三级视图：

1. **教室列表**：当前校区所有教室的表格概览。
2. **教室详情**：单教室的 4 个 Tab 页（总览、终端列表、桌面与存储、告警）。
3. **终端详情**：从教室详情的终端列表进入，独立文件见 [WEB-02A_终端详情.md](./WEB-02A_终端详情.md)。

导航路径：WEB-02 教室列表 → 教室详情 → 终端详情。每一级在面包屑中独立展示。

## 3. 导入终端清单向导

点击"导入终端清单"进入向导面板：

```text
┌──────────────────────────────────────────────────────────────┐
│ ← 返回教室列表                                               │
│                                                              │
│ 导入终端清单                                                  │
│ 选择母机导出的教室终端清单文件。清单包含终端信息和桌面包，     │
│ 系统将创建教室、登记终端并导入桌面镜像和快照文件。             │
│                                                              │
│ ┌── 文件选择区 ──────────────────────────────────────────┐   │
│ │  选择教室终端清单文件（.xlsx，由终端侧「导出清单」生成） │   │
│ └────────────────────────────────────────────────────────┘   │
│                                                              │
│ 导入目标：[创建新教室 ▼]                                      │
│                                                              │
│ ── 新教室信息 ──────────────────────────────────             │
│ 教室名称：[从清单自动读取_________]                           │
│ 位置：    [如：创意楼B座 3F________]                          │
│ 备注：    [可选___________________]                           │
│ ──────────────────────────────────────────                   │
│                                                              │
│ [确认导入]  [取消]                                            │
│                                                              │
│ 导入进度：████████████░░░░░░░░░░░░  52%                      │
│ 正在导入桌面镜像与快照文件…（3/5 桌面包）                     │
└──────────────────────────────────────────────────────────────┘
```

### 3.1 导入目标

- **创建新教室**：使用清单中的教室名称（可修改），创建教室并登记全部终端。
- **追加到已有教室**：终端追加到已选教室，已有终端不受影响。

### 3.2 导入过程

教室终端清单包含终端信息和桌面包（PRD FR-04）。桌面包内含实际的桌面镜像和快照文件，因此导入是耗时操作。

- 导入过程展示进度条和当前阶段（读取终端信息 → 导入桌面镜像与快照文件 → 登记终端 → 对齐在线状态）。
- 导入完成后显示"已成功导入终端清单 — N 台终端 · M 个桌面"。
- 导入后自动开始对齐终端在线状态。

| 指标 | 说明 |

| 分组 | 操作 | 前提 | 说明 |
| :--- | :--- | :--- | :--- |
| 单教室操作 | 批量关机 | 选中在线终端 | 向选中终端发送关机指令（见 4.3.3） |
| 单教室操作 | 批量重启 | 选中在线终端 | 向选中终端发送重启指令（见 4.3.4） |
| 单教室操作 | 单教室桌面部署 | 选中在线终端 | 当前教室内桌面部署（见 4.3.5） |
| 单教室操作 | 修改 IP | 选中在线终端 | 批量修改终端 IP 地址（见 4.3.7） |
| 单教室测试 | 网络测试 | 选中在线终端 | 终端网络连通性测试（见 4.3.8） |
| 多教室操作 | 跨教室桌面部署 | 无需预选终端 | 向其他教室任意终端部署桌面（见 4.3.6） |

### 4.4 桌面与存储 Tab（只读）

标注为"只读视图 — 桌面数据来自教室终端清单导入和终端自行上报"。

展示内容：

1. 教室统计：桌面数、快照数、镜像数、数据盘数。
2. 桌面目录卡片：每张桌面显示名称、操作系统、数据盘配置、备注、创建/更新时间。
3. 使用终端列表：展示该教室内哪些终端使用了该桌面（座位号、机器名、IP、在线状态），可点击跳转 WEB-02A 终端详情。
4. 镜像快照文件链：以树形结构展示 [镜像] → [快照] → [快照] 的派生关系。镜像和快照属于内部资产视图，展示供运维人员了解快照层级和存储分布，不可操作。
5. 数据盘汇总表：所属桌面、数据盘配置。

说明：教室级桌面视图以教室为上下文，能细化到每个桌面被教室内哪些终端使用。平台级桌面资产管理见 WEB-03。

### 4.5 告警 Tab

展示当前教室的活跃告警列表：

- 严重度摘要：高/中/低，各多少条。
- 支持按严重度或时间排序切换。
- 每条告警：严重度标签、标题、详情、来源终端信息（可点击跳转终端详情）、时间。

空态：无活跃告警时显示"当前无活跃告警"。
