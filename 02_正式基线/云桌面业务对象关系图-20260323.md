# 云桌面业务对象关系图

<a id="diagram-notes"></a>
## 图示说明

- 本版保留较完整的业务对象与关键数据流转，供详细讨论使用。
- 教师终端与学生终端共用同一终端模型，通过终端用途标记区分。
- 基础镜像与快照随桌面按需同步。
- 交付测试可由临时母机当前教室界面或服务器平台发起，测试结果与告警自动同步。
- 图中的关系说明用于帮助理解业务语义，不要求逐字落到界面文案。

<a id="diagram"></a>
## 业务对象关系图

```mermaid
classDiagram
    %% ================= 服务器域 =================
    namespace 服务器域 {
        class 服务器 {
            网络地址：IP或域名
            系统运行负载
            授权许可点数
        }
        class 管理员账号 {
            账号类型：系统管理员或校区子管理员
            可见校区范围
        }
        class 教室 {
            教室名称
            座位布局
        }
        class 服务器端终端登记 {
            MAC地址列表：多网卡
            当前工作MAC
            绑定的IP与计算机名
            归属的教室与座位
            终端用途标记：教师用途、学生用途或其他已登记用途
            实时在线状态
        }
        class 服务器端告警记录 {
            告警类型
            告警源设备：终端或服务器自身
            溯源教室与座位
            处理状态
        }
        class 服务器端基础镜像 {
            OS类型
        }
        class 服务器端快照 {
            快照名称
        }
        class 服务器端测试视图 {
            测试类型与目标范围
            平台查看与导出结果
        }
    }

    %% ================= 辅助工具域 =================
    namespace 辅助工具域 {
        class 安装U盘 {
            终端底层系统安装包
            基础镜像包
        }
        class 教室终端清单 {
            单教室全体终端信息与排布关系
            每台终端MAC地址列表与当前工作MAC
        }
        class 桌面包 {
            桌面基本信息
            大文件
        }
    }

    %% ================= 终端基座域 =================
    namespace 终端基座域 {
        class 终端 {
            MAC地址列表：多网卡
            当前工作MAC
            硬件能力规格
            外设连接状态
        }
        class 终端业务配置 {
            分配的局域网IP
            分配的计算机名
            所属教室与座位
            终端用途标记：教师用途、学生用途或其他已登记用途
        }
        class 终端运行状态 {
            网络在线状态
            CPU与内存负载
        }
        class 本地告警记录 {
            告警类型
            处理状态
        }
        class 本地测试结果 {
            测试类型
            结果摘要
            生成时间
        }
        class 本地基础镜像 {
            OS类型
        }
        class 本地快照 {
            快照名称
        }
    }

    %% ================= 终端BIOS域 =================
    namespace 终端BIOS域 {
        class BIOS引导策略 {
            可见启动项列表
            默认启动项
            数据还原模式：系统盘还原或数据盘保留
        }
    }

    %% ================= 终端桌面域 =================
    namespace 终端桌面域 {
        class 桌面 {
            部署方式：快照式或物理分区式
            桌面数据同步状态：上传中或已就绪
        }
        class 教师机桌面 {
            教室开关机能力：唤醒或关机
        }
    }

    服务器 "1" *-- "1..n" 管理员账号
    管理员账号 ..> 教室 : 按校区范围可见与管理
    服务器 "1" *-- "0..n" 教室
    教室 "1" *-- "0..n" 服务器端终端登记
    服务器 "1" o-- "0..n" 服务器端基础镜像
    服务器 "1" o-- "0..n" 服务器端告警记录
    服务器 "1" o-- "0..n" 服务器端测试视图
    服务器端基础镜像 <|-- 服务器端快照

    终端 "1" o-- "0..n" 本地基础镜像
    终端 "1" *-- "1" 终端业务配置
    终端 "1" *-- "1" 终端运行状态
    终端 "1" o-- "0..n" 本地告警记录
    终端 "1" o-- "0..n" 本地测试结果
    本地基础镜像 <|-- 本地快照
    桌面 <|-- 教师机桌面 : 派生

    %% 交付登记与导出导入
    终端 ..> 教室终端清单 : 汇总导出教室终端完整信息
    教室终端清单 ..> 服务器端终端登记 : 导入平台创建教室与登记终端
    教室终端清单 o-- 桌面包 : 包含

    %% 初始安装与桌面资产同步
    安装U盘 ..> 终端 : 安装云桌面管理系统
    安装U盘 ..> 本地基础镜像 : 提供初始镜像来源
    本地基础镜像 <..> 服务器端基础镜像 : 随桌面按需同步
    本地快照 <..> 服务器端快照 : 随桌面按需同步

    %% 资产同步与状态对齐
    终端运行状态 ..> 服务器端终端登记 : 上传
    终端业务配置 <..> 服务器端终端登记 : 双向同步
    终端 ..> 本地告警记录 : 自动采集/比对生成
    本地测试结果 ..> 本地告警记录 : 异常时生成告警
    本地测试结果 <..> 服务器端测试视图 : 当前教室或平台发起/查看
    本地告警记录 <..> 服务器端告警记录 : 自动同步展示

    %% 引导、环境注入与管控
    BIOS引导策略 --> 桌面 : 设置加载目标与还原模式
    本地快照 ..> 桌面 : 作为快照式桌面的数据来源
    终端业务配置 ..> 桌面 : 应用教室与网络身份
```
