<a id="diagram-notes"></a>
## 图示说明

- 本版为简化阅读版，优先呈现主业务对象与主数据流转。
- 教师终端与学生终端共用同一终端模型，通过终端用途标记区分。
- 基础镜像与快照随桌面按需同步。
- 交付测试可由临时母机当前教室界面或服务器平台发起，测试结果与告警自动同步。
- 更细的字段说明与机制解释，请结合 [云桌面业务对象关系图-20260323.md](云桌面业务对象关系图-20260323.md) 和 PRD 正文阅读。

<a id="diagram"></a>
## 业务对象关系图

```mermaid
classDiagram
    namespace 服务器域 {
        class 服务器 {
            网络地址：IP或域名
            系统运行负载
            授权许可点数
        }
        class 管理员账号 {
            账号类型：系统管理员或校区子管理员
            可见校区范围
        }
        class 教室 {
            教室名称
            座位布局
        }
        class 服务器端终端登记 {
            教室与座位
            身份与网络标识
            终端用途标记
            实时在线状态
        }
        class 服务器端告警记录 {
            告警类型
            告警来源
            处理状态
        }
        class 服务器端测试视图 {
            测试类型与目标范围
            平台查看与导出结果
        }
        class 服务器端基础镜像 {
            OS类型
        }
        class 服务器端快照 {
            快照名称
        }
    }

    namespace 辅助工具域 {
        class 安装U盘 {
            终端底层系统安装包
            基础镜像包
        }
        class 教室终端清单 {
            单教室终端信息与排布关系
        }
        class 桌面包 {
            桌面基本信息
            大文件
        }
    }

    namespace 终端域 {
        class 终端 {
            当前工作MAC
            MAC地址列表
            硬件与外设状态
        }
        class 终端业务配置 {
            教室与座位
            IP与计算机名
            终端用途标记
        }
        class 终端运行状态 {
            在线状态
            资源负载
        }
        class 本地告警记录 {
            告警类型
            处理状态
        }
        class 本地测试结果 {
            测试类型
            结果摘要
        }
        class 本地基础镜像 {
            OS类型
        }
        class 本地快照 {
            快照名称
        }
        class BIOS引导策略 {
            启动项列表
            默认启动项
            还原模式
        }
        class 桌面 {
            部署方式：快照式或物理分区式
            桌面同步状态：上传中或已就绪
        }
        class 教师机桌面 {
            教室开关机能力
        }
    }

    服务器 "1" *-- "1..n" 管理员账号
    管理员账号 ..> 教室 : 按校区范围可见与管理
    服务器 "1" *-- "0..n" 教室
    教室 "1" *-- "0..n" 服务器端终端登记
    服务器 "1" o-- "0..n" 服务器端告警记录
    服务器 "1" o-- "0..n" 服务器端测试视图
    服务器 "1" o-- "0..n" 服务器端基础镜像
    服务器端基础镜像 <|-- 服务器端快照

    终端 "1" *-- "1" 终端业务配置
    终端 "1" *-- "1" 终端运行状态
    终端 "1" o-- "0..n" 本地告警记录
    终端 "1" o-- "0..n" 本地测试结果
    终端 "1" o-- "0..n" 本地基础镜像
    本地基础镜像 <|-- 本地快照
    桌面 <|-- 教师机桌面 : 派生

    终端 ..> 教室终端清单 : 汇总导出教室终端完整信息
    教室终端清单 ..> 服务器端终端登记 : 导入平台创建教室与登记终端
    教室终端清单 o-- 桌面包 : 包含
    安装U盘 ..> 终端 : 安装云桌面管理系统
    安装U盘 ..> 本地基础镜像 : 提供初始镜像来源
    本地基础镜像 <..> 服务器端基础镜像 : 随桌面按需同步
    本地快照 <..> 服务器端快照 : 随桌面按需同步
    终端运行状态 ..> 服务器端终端登记 : 上传实时状态
    终端业务配置 <..> 服务器端终端登记 : 双向同步业务配置
    本地测试结果 <..> 服务器端测试视图 : 当前教室或平台发起/查看
    本地告警记录 <..> 服务器端告警记录 : 自动同步展示
    BIOS引导策略 --> 桌面 : 配置
    本地快照 ..> 桌面 : 制作桌面
```
