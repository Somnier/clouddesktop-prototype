# WEB-01 平台总览

> 页面角色：平台侧第一落点，当前校区的运行概览。
> 执行主体：系统管理员或校区子管理员。
> 数据来源：服务器端登记信息、终端上报运行状态、告警记录、任务进度。

## 1. 页面职责

平台总览是管理员登录后的默认页面。目标是在一屏之内让管理员判断当前校区的整体健康状况，并快速定位到需要关注的教室或异常。

## 3. 指标网格

指标网格展示当前校区的核心运行数据：

| 指标 | 主值 | 辅助值 |
| :--- | :--- | :--- |
| 教室数 | 当前校区教室总数 | 已创建教室数 |
| 注册终端 | 终端总数 | 教师终端数 · 学生终端数 |
| 在线率 | 在线终端 / 总终端 | 在线数 · 离线数 |
| 活跃告警 | 当前未处理告警数 | 有告警时红色高亮 |
| 桌面数 | 校区桌面总数 | 服务器存储使用率 |

点击指标卡片可跳转到对应页面：告警 → WEB-04，教室数 → WEB-02，桌面 → WEB-03。

## 4. 服务器状态卡片

展示当前校区所属服务器的基础信息和资源占用：

- 服务器名称、内网 IP、外网 IP、SSL 状态。
- CPU、内存、存储使用率仪表。
- 使用率超过 80% 时红色警示，60%~80% 时黄色提示。

## 5. 任务进度

> 参考：PRD FR-12 第 4 条。

展示当前正在执行的长任务（教室终端清单导入、桌面上传、跨教室桌面部署、一键替换、一键重置等）：

| 列 | 说明 |
| :--- | :--- |
| 任务类型 | 教室终端清单导入 / 桌面上传 / 跨教室桌面部署 / 一键替换 / 一键重置 |
| 目标 | 教室名称或终端信息 |
| 状态 | 排队 / 执行中 / 阻塞 / 失败 |
| 进度 | 已完成/总数，百分比 |
| 开始时间 | 任务发起时间 |

- 阻塞或失败的任务红色高亮。
- 无进行中任务时显示"无执行中任务"。
- 点击任务行可跳转到对应教室详情或终端详情。

## 6. 异常与风险

> 参考：PRD FR-12 第 5 条。

展示当前校区的异常概况：

- 未处理告警数，按高/中/低分布。
- 按教室分布的告警摘要（哪些教室有告警、各多少条）。
- 高频故障终端或高风险教室（多条告警集中的终端或教室）。
- 点击告警数可跳转 WEB-04 告警中心，点击教室名可跳转 WEB-02 教室详情告警 Tab。

## 7. 教室概况表

展示当前校区所有教室的摘要信息：

| 列 | 说明 |
| :--- | :--- |
| 教室 | 教室名称，可点击跳转 WEB-02 教室详情 |
| 位置 | 楼栋 + 楼层 |
| 阶段 | 空白 / 已部署 |
| 终端数 | 教室内注册终端总数 |
| 在线 | 在线/总数 |
| 告警 | 活跃告警数，有告警时红色 |
| 任务 | 当前执行中的任务名称，无则显示"-" |

## 8. 最近日志

按时间倒序展示最近 8 条系统日志：

- 每条日志包含等级标签（信息/警告/错误）、摘要文本和时间。
- 日志内容来自服务器端记录。

## 9. 空态

- 校区无教室时：指标归零，教室概况显示"暂无教室，请先通过教室管理导入终端清单"。
- 服务器信息未配置时：服务器状态卡片显示"未连接"。
