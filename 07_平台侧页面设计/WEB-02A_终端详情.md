# WEB-02A 终端详情

> 页面角色：从 WEB-02 教室详情的终端列表进入的单终端全量信息页。
> 执行主体：系统管理员或校区子管理员。
> 数据来源：服务器端登记信息、终端上报的运行状态和桌面资产、告警记录。

## 1. 页面职责

终端详情页是平台侧查看单台终端全量信息的入口。所有数据均为只读展示，来自终端自行上报和服务器端登记信息。平台不在此页修改终端配置。

## 3. 信息区域

### 3.1 配置

| 字段 | 说明 |
| :--- | :--- |
| 机器名 | 终端名称。未命名时显示"未命名" |
| 座位号 | 未分配时显示"未分配" |
| 终端用途 | 教师终端 / 学生终端 / 未设定 |
| IP 地址 | 未配置时显示"未配置" |
| 子网掩码 | 默认 255.255.255.0 |
| 网关 | 未配置时显示"未配置" |
| DNS | 未配置时显示"未配置" |

### 3.2 外部连接

| 字段 | 说明 |
| :--- | :--- |
| 服务器地址 | 终端配置的服务器地址 |
| 服务器状态 | 已连接 / 未配置 |
| 同步状态 | 未同步 / 同步中 / 已同步 / 同步失败 |
| 最近同步 | 最近一次同步的相对时间 |

### 3.3 硬件信息

| 字段 | 说明 |
| :--- | :--- |
| 网卡列表 | 所有 MAC 地址，当前工作 MAC 标记"工作中" |
| 处理器 | CPU 型号 |
| 显卡 | GPU 型号 |
| 内存 | 容量和规格 |
| 硬盘 | 型号 + 出厂 SN 码 |

### 3.4 运行状态

| 字段 | 说明 |
| :--- | :--- |
| CPU | 使用率 + 温度。超 80% 或超 80°C 时红色警示 |
| GPU | 使用率 + 温度。同上 |
| 内存 | 已用/总量 + 百分比 |
| 磁盘 | 已用/总量 + 百分比 |
| 最近心跳 | 相对时间 |

### 3.5 已部署桌面列表

展示终端上报的全部桌面信息。每个桌面条目包含：

| 字段 | 说明 |
| :--- | :--- |
| 桌面名称 | 用户侧名称 |
| 操作系统 | OS 类型与版本 |
| 默认启动 | 是否为默认启动桌面 |
| 物理部署 | 是否为物理分区部署 |
| 隐藏 | 是否在启动列表中隐藏 |
| 还原模式 | 系统盘还原 / 数据盘保留 等 |
| 数据盘 | 盘符/大小/格式 |
| 创建时间 | 桌面创建时间 |
| 更新时间 | 最近编辑时间 |

物理部署桌面标注"仅本机安装，服务器无此桌面"。

### 3.6 当前任务

若终端正在参与某个批量任务，展示任务信息：

- 任务名称、状态（排队中/传输中/写入中/重启中/已完成/失败）。
- 失败时显示失败原因。

### 3.7 告警

展示该终端的活跃告警列表，格式与 WEB-04 告警中心一致。

## 4. 空态

- 未部署终端：配置区域大部分字段显示"未命名""未配置""未分配"，桌面列表为空。
- 离线终端：运行状态区域显示最后一次上报数据，心跳时间标记为较久前。
